# Assignment 1 - Smart Contract Development

## Objective

Develop and compile a simple Solidity smart contract.

## Included Files

- `contract.sol` - Main smart contract file
- `screenshots/` - Add compilation and deployment screenshots here

## Contract Purpose

This contract stores simple product purchase records on-chain. Each purchase saves the product name, price, and buyer wallet address.

## Main Functions

- `buyProduct(string _name, uint256 _price)` - Stores a product purchase
- `getProductCount()` - Returns total number of recorded products

## Compilation And Deployment Steps

1. Open Remix IDE.
2. Create or import `contract.sol`.
3. Select Solidity compiler `0.8.x`.
4. Compile the contract successfully.
5. Deploy using JavaScript VM or MetaMask testnet account.

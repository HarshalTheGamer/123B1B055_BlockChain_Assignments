# Assignment 2 - Polygon Deployment

## Objective

Deploy the Solidity smart contract on Polygon testnet.

## Included Files

- `contract.sol` - Solidity contract used for deployment
- `screenshots/` - Add deployment and explorer screenshots here

## Network Details

- Network: Polygon Amoy Testnet
- Chain ID: `80002`
- Currency Symbol: `POL` or `MATIC` depending on wallet display
- RPC URL: `https://rpc-amoy.polygon.technology/`
- Explorer: `https://amoy.polygonscan.com/`

## Deployment Steps

1. Open Remix IDE and load `contract.sol`.
2. Connect Remix to MetaMask.
3. Switch MetaMask to Polygon Amoy.
4. Compile the contract.
5. Deploy the contract and confirm the transaction in MetaMask.
6. Open the transaction or contract on PolygonScan.

DAO Workflow

1

Deploy core contracts

Deploy DonationContract.sol first, then deploy GovernanceDAO.sol using the donation contract address.

2

Authorize DAO control

Call setAuthorizedDAO() so approved proposals can trigger fund release safely from the DAO contract.

3

Grant tokens and create proposal

Distribute governance tokens, choose an active campaign, then create a proposal with a snapshot of voter balances.

4

Vote and execute

After quorum and deadline, execute the proposal. If it passes, the beneficiary receives campaign funds on-chain.
